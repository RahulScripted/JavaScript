How would you add a hyphen between two strings "first" and "second" and store their concatenation in variable named x ? Select correct options from the following :



1.  x = "first+second";

2.  x = "first" + "-" + "second";

3.  x = "first".concat("-", "second");

4.  x = "first" . "-" . "second"




Answers: 

2.  x = "first" + "-" + "second";

3.  x = "first".concat("-", "second");
