How are parameters and arguments related in a function? Select one of the following options:

1.  Parameters are values passed to the function, and arguments are defined inside the function

2.  Parameters are defined inside the function, and arguments are values passed to the function

3.  Both terms refer to the same thing

4.  Neither parameters nor arguments are used in the function





Answer: 2.  Parameters are defined inside the function, and arguments are values passed to the function
